lib_mp3
============

Summary
-------

MP3 decoder library. Uses Underbit Technolgies' MAD mpeg audio
decoder, which is under a GPLv2 license.

Features
........

.. sidebysidelist::

 * MP3 Decoding at 16 and 24 bit.

Resource Usage
..............

.. resusage::

 * Single thread
 * 56kb memory
